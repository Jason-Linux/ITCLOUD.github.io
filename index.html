Contexte ITCLOUD

  • L’entreprise ITCLOUD à pour secteur d’activité l’infogérance, l’entreprise LIBRE EN VOL requière les services proposés par l’entreprise ITCLOUD afin de manager a distance l’AD de libre en vol ainsi que d’installer une base de connaissance pour mutualiser la documentation, de plus l’entreprise ITCLOUD héberge le Site de LIBRE EN VOL qui est disponible depuis la DMZ d’ITCLOUD.

  • Besoin d’ITCLOUD : Windows Server pour l’entreprise un serveur pour le stockage ainsi qu’une base de données, un système de ticket, Un outil de supervision (EON), un système de filtrage (PFSENSE). Une DMZ : Un Docker pour héberger le site web de LIBRE EN VOL, un serveur web avec WordPress pour héberger le site d’ITCLOUD ainsi que le système de ticket et la base de connaissance, un Windows Server qui seras le contrôleur principal du domaine libre en vol.

  • Besoin de LIBRE EN VOL : En local : un poste client, un pare feu pour filtrer le trafic ainsi qu’un serveur RODC en lien avec le Windows server disponible dans la DMZ de l’entreprise ITCLOUD
